# API Escala

API desenvolvida em ASP.NET Core para gerenciamento de escalas.

## Tecnologias Utilizadas

- C#
- ASP.NET Core
- Entity Framework Core
- SQL Server
- Swagger / OpenAPI
- React (frontend em desenvolvimento)

## Funcionalidades

- Cadastro de participantes
- Cadastro de funções
- Criação de escalas
- Distribuição automática de escalas
- Distribuição manual de escalas
- Consulta de escalas

## Como Executar o Projeto

### Clone o repositório

```bash
git clone http://github.com/Luciano1010/api-escala.git
```

### Entre na pasta do projeto

```bash
cd escalaApi
```

### Execute a aplicação

```bash
dotnet run
```

## Documentação da API

Após executar o projeto, acesse o Swagger:

```txt
http://localhost:5280/swagger
```

A documentação da API é gerada automaticamente pelo Swagger/OpenAPI.
